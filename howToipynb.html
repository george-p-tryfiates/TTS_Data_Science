From Command Line

    1. Open Anaconda and enter 'jupyter notebook'. This pulls up jupyter notebook in your browser showing your computer's directories
    2. Navigate to git repo clone directory in jupyter notebook
    3. Create a Python 3 Notebook in the directory and save your notebook
    4. Open computer command line (I use GitBash)
    5. Navigate to git repo directory
    6. Enter "git status" to see whether the repo is committed & synced
    7. Enter "git add ." to add all non-committed files
    8. Enter "git commit -a" to commit the files in the staging area to the main branch
    9. Enter "git push origin" to update the online repo
    10. You are done!

    With GitHub Visual Interface

    1. Create Python 3 Notebook in jupyter notebook and save it in your desktop repo
    2. Open browser and navigate to Github Repo
    3. Click "Add File" and then "Upload File"
    4. Click on the Notebook you want to upload and click upload
    5. You are done!
